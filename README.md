## Steps to setup the starter template

1. Clone the project from GitHub

```
git clone https://github.com/Olawumi-Olusegun/Node-Express-Typescript-Starter.git <ProjectName>
```

2. Move into the folder structure

```
cd <ProjectName>

```

3. Install npm dependencies

```
npm install
```

4. Create a new .env file in the root directory and add PORT

```
echo PORT=3001 >> .env
```

5. Start express server

```
npm run dev
```
